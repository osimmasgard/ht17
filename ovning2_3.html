Övning 2.3

Boxmodellen

HTML-element kan liknas vid lådor och det är dessa lådor man med hjälp av CSS flyttar runt i förhållande till varandra för att bygga upp designen av en HTML-sida. Det är detta som Boxmodellen syftar på. Boxen runt ett element består av marginaler, kanter, paddning.

Exempel

Kantlinjer

Kantlinjer styr en kantmarkering runt elementet. Attributet "border" är ett komposit attribut i vilket du kan lista en rad olika egenskaper så som
  • border-color (kantens färg)
  • border-style (om den ska vara heldragen, punkter, små korta linjer etc)
  • border-width (som används för att sätta tjockleken på kanten)

Paddning

Attributet "padding" styr med hjälp av måttenheterna (se övning 2.1) avstånden till toppen, botten, höger och vänster. Exempel:

span.padding {

padding-top: 50px;
padding-right: 30px;
padding-bottom: 50px;
padding-left: 80px;
}

Synlighet

Med hjälp av "visibility" så kan man ändra om elementet ska synas eller inte, bland annat. Attributet "clip-path" gör så att man kan endast välja att visa en del av ett element ska synas, och att resten ska vara osynligt. Attributet "display" bestämmer "boxens" förhållande gentemot omkringliggande element. Använder man värdet "block" så beter sig boxen som friliggande, med tomma rader innan och efter. "Inline" betyder att det ligger i samma nivå som omliggande element. Det finns även andra värden man kan laborera med. Overflow: detta attribut bestämmer vad som ska hända med elementet om det är för stort för att visas inom en viss area (liten displayport eller föminskat fönster). Man kan exempelvis sätta värdet "scroll" som i dessa fall lägger till en scrollbar till elementet. Eller så väljer man att dölja den del som inte syns med "hidden".

Transparens etc

Här är en box som är delvis transparent, har rundade hörn och multipla bakgrundsbilder, den har även en skugga i rött.
div.weirdbox {
margin-bottom: 2em;
padding: 50px;
background: url("../images/poster.png"), url("../images/fot.png");
border-radius: 5px;
box-shadow: 10px 10px 5px #b93030;
opacity: 0.7;
}

Här är en box som använder en gradient bakgrund

div.weirdbox2 {
margin-bottom: 2em;
padding: 50px;
background: linear-gradient(red, yellow);
border-radius: 5px;
}
