Övning 2.4

Background

Används för att styra bakgrunden på en webbsida. Precis som i tidigare exempel så finns ett antal specifika inställningar som kan göras, nämligen:
  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position
men man kan använda det sammansatta begreppet "background" för att styra de på en och samma rad (ordningen ovan gäller). Exempelvis:

span.padding {
background: #ffffff url("fot.png") no-repeat right top;
}

CSS och img-taggen!

Antingen använder man en class-selektor för att styra hur img-taggen ska se ut/fungera mha CSS, eller så styr man alla img-taggar som så:

img {

border: 1px solid #ddd;
border-radius: 4px;
padding: 5px;
width: 150px;
}
