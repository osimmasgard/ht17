Övning 2.5

Länkar!

Länkar kan ha olika tillstånd:
  • a:link - en länk som användaren inte klickat på än
  • a:visited - en länk som användaren klickat på
  • a:hover - en länk som väljs när användaren rör musen på den
  • a:active - en länk som användaren redan klickat på
Jag kommer att ha följande inställningar satta i CSS-filen.

a:active { background-color: green;
cursor: crosshair; }

a:hover { background-color: yellow; }

a:link { font-style: oblique;
font-family: Courier New, Courier, monospace;
font-size: 0.8em; }

a:visited { font-weight: bolder;
}
